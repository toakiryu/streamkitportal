## ステップ１

[Discord StreamKit Overlay](https://streamkit.discord.com/overlay) というサイトにアクセスします。

![step-1 image](/wp-content/discord/docs/overlay/widget-setup/step-1.png)

## ステップ２

ステータスウィジェット, チャットウィジェット, ボイスウィジェット から使用するものを選択してください。

![step-1.5 image](/wp-content/discord/docs/overlay/widget-setup/step-1-2.png)

それぞれ設定項目は違いますが、わかりやすいプレビューもついているので頑張って設定してください。

※この画像はステータスウィジェットです

![step-2 image](/wp-content/discord/docs/overlay/widget-setup/step-2.png)

## ステップ３

簡単な設定が完了したら、プレビューの下にある URL をコピーしてください。

![step-3 image](/wp-content/discord/docs/overlay/widget-setup/step-3.png)

## ステップ４

OBS を起動して、任意のシーンを選択しソースの追加ボタンを押してください。

![step-4 image](/wp-content/discord/docs/overlay/widget-setup/step-4.png)

## ステップ５

`ソース > ブラウザ` を選択してください。

![step-5 image](/wp-content/discord/docs/overlay/widget-setup/step-5.png)

## ステップ６

必要であれば名前を設定し `OK` を押してください。

![step-6 image](/wp-content/discord/docs/overlay/widget-setup/step-6.png)

## ステップ７

プロパティー設定が開いたことを確認して、一番上の `URL` の入力欄に、[ステップ３](#ステップ３)でコピーした URL を入力してください。

![step-7 image](/wp-content/discord/docs/overlay/widget-setup/step-7.png)

## ステップ８

下にスクロールして、`カスタムCSS` という項目を見つけてください。ここでは任意の CSS コードを入力することで、CSS を適応させることができます。

![step-8 image](/wp-content/discord/docs/overlay/widget-setup/step-8.png)

## ステップ９

最後に `OK` を押して実装完了です。

![step-9 image](/wp-content/discord/docs/overlay/widget-setup/step-9.png)
